WHAT IS A REQUEST?

An HTTP request is created by a client which is sent to the server to be able to access the web resources. The format of the HTTP request shown bellow:

  1. Request line:
  2. Header fields
  3. Message Body

Request line

This component is cotnsist of three elements. The first part is the method. It is a verb (e.g. GET, PUT,POST) or a noun (e.g. HEAD) that informs the server what it as to do with the resource. For example, POST is used to create a resource. The second part of request line is the path. This component recognises the resource on the server. The third element is HTTP version number which displays the HTTP specification.

Header Fields

The header fields enables the client to provide further information regarding the request and client to the server.

Message Body

The message body is also known as the entity body which is the actual content of the message. POST method requests has message body containing the data to the server. Whereas for GET, GET request method does not have a message body as it is only retrieving resource.

Status Error Codes

If something wrong happens in the server while processing the request, the client will be informed through the display of a HTTP status code as a response. Not all status codes signify error in the server. However, we will focus on knowing some common error codes.

Error name Why
301 Permanently Moved A 301 response shows that a distinct webpage is permanently moved to a different URL
401 Unauthorized The response means that the request is not complied as valid authentication credentials for the resource is insufficient or invalid
404 Not Found This error indicates that page the user is searching for does not exist or cannot be found. This occurence could be caused by entering an incorrect URL or the page has been deleted.
410 Gone 401 error indicates that the requested resource has been permanently removed from the internet.
500 Internal Server Error This error depicts that there is a general issue with the website's server.
502 Bad Gateway This issue occurs when one server on ther internet gets an invalid response from another server. This error could be fixed by refreshing the browser or clearing the browser site.
