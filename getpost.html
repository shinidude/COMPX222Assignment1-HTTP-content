HTTP REQUEST TYPES

GET and POST

There are a list of HTTP request method which describes what action is needed to be done to the requested resource. There are two main HTTP request methods: GET and POST.

GET is implemented to request for a resource (e.g. hypertext document and images). This method only employed for retrieving data and it does not alter any resourcein the server. Get should not get involved in any sensitive (e.g. password, financial information, security number) information. This information may become visible in the URL and also can become a cache data. POST, on the other hand, is utilised to send information to the web server to modify a resource. POST is more secure than GET as the query parameters are not displayed on the URL and would less likely be cache. Here are some examples of GET and POST
